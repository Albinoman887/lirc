Welcome to the LIRC Manual

LIRC (Linux Infrared Remote Control) gives you the possibility to control your Linux box with a remote control. You can use it with applications under X-Windows, or to control a application in a pseudo-terminal. Emulated mouse support is also included. All what you need is an IR-Receiver [on serial or parallel port], available as a ready-to-run-device with remotes like the Animax Remote Control and the Irman or a device build with your own hands ;) But that's not all, experimental IR sending support is also included, this means you can send IR-Commands and control your TV, VCR ....

karsten, 1999-FEB-14

Since the pre0.5.5 Version of LIRC, there is also support for different TV-Cards: Hauppauge, Avermedia, Fly98. This support is in alpha stage don't expect a full working driver, the most problems discussed in the mailing list are related to this cards. Whether this drivers are included in the upcoming lirc-0.5.5 release depends from the drivers itself. If there are to many problems, the drivers will only be available in the CVS Tree or are removed completely if nobody develops them anymore.

LIRC doesn't depend on special remotes, it is able to 'understand' new remotes by learning their codes. This is done with the irrecord program

karsten, 1999-SEP-04
